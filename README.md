# ddos-scapy
A python3 project on a ddos attack script totally anonymous with scapy

Usage:

$ sudo python3 -m pip install scapy
$ sudo python3 ddos.py <targetIP> <targetPort>

Exemple:

$ sudo python3 ddos.py google.com 443
